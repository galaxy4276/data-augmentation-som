### 현재 상황과 과제

- 현재 매칭 데이터가 매우 적음(상호 좋아요 320건, 채팅 진행 112건)
- 데이터 부족으로 인해 학습에 한계가 있음
- 모델 평가를 위한 테스트셋 구성 필요

### 데이터 증강 계획

- 기존 320건의 매칭 데이터를 클러스터링하여 약 30개 유형으로 분류
- 각 클러스터별로 ChatGPT를 활용해 100-500개의 유사 데이터 생성
- 최종적으로 3,000-15,000개의 학습 데이터 구축 목표
- 실제 데이터 300개는 테스트셋으로, 20개는 검증셋으로 활용

### 프로필 이미지 생성

- 생성된 텍스트 프로필에 맞는 가상 인물 이미지 생성
- 이미지 생성 서비스를 활용하여 남성/여성 프로필 사진 제작

### 모델 학습 및 접근법
