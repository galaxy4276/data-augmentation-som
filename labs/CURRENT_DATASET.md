## 연구 개요
본 보고서는 실제 운영 중인 데이팅 앱의 데이터베이스를 분석하여, 이미지-특징정보-이상형의 삼중쌍 데이터셋 구축 가능성을 평가한 연구 결과입니다.

채팅 기능이 도입된지 1달밖에 지나지않아 다른 지표(상호 간 좋아요 수행)를 성공 기준으로 하여, 실제 매력 패턴이 반영된 학습 데이터셋의 잠재력을 분석하였습니다.

## 2. 연구 데이터셋 구성 분석
### 2.1 데이터셋 유형별 규모
#### 기본 페어 데이터셋
- **정의**: 이미지 + 프로필 기본 정보 + 선호도
- **규모**: 3,726쌍
- **활용**: 기본 스타일 분류 및 시각 특징 학습
#### 상호 매력 페어 데이터셋 ⭐ **핵심**
- **정의**: 이미지 + 이상형정보 + **상호 좋아요 성공**
- **규모**: 282쌍
- **가치**: 실제 양방향 매력이 검증된 데이터
- **특징**: 평균 13.9개 이미지/페어, 100% 프로필 연결
#### 확장 잠재 데이터셋
- **최대 규모**: 5,328쌍 (모든 이미지 보유자)
- **확장 방법**: 자동 선호도 생성 기술 적용
### 2.2 상호 좋아요 기반 성공 패턴
#### 성공 매칭 특징 분석
- **성공률**: 282쌍 / 5,561,568쌍 = 0.005%
- **상호작용 깊이**: 평균 13.9개 이미지 교환
- **선호도 일치도**: 높은 MBTI/스타일 선호도 일치 패턴
#### 후보 Pool 분석
- **남성 입장**: 평균 1,262명의 나이 적합 여성 후보
- **선택의 폭**: 1,413명의 전체 여성 Pool
- **경쟁률**: 약 0.08% (282성공/1,413후보)

## 5. 학습 데이터셋 구성 제언
### 5.1 3단계 데이터셋 구조
#### 1단계: 기반 데이터셋 (3,726쌍)
- **구성**: 이미지 + 기본 프로필 + 선호도
- **목적**: 기본 스타일 분류 모델 학습
- **활용**: 시각 특징-선호도 기본 연관성
#### 2단계: 검증 데이터셋 (282쌍) ⭐
- **구성**: 이미지 + 이상형 + 상호 좋아요 성공
- **목적**: 예측 모델 실제 성능 검증
- **활용**: 상호 매력 예측 정확도 측정
#### 3단계: 확장 데이터셋 (1,000쌍)
- **구성**: 실제 282쌍 + 생성 718쌍
- **목적**: 대규모 모델 학습
- **활용**: 상용화 수준 예측 모델
### 5.2 ChatGPT 기반 데이터 증강 전략
#### 증강 대상
1. **이상형 기반 시뮬레이션**: 사용자 선호도에 맞는 가상 프로필
2. **스타일 변형 생성**: 기존 이미지의 스타일 변형
3. **실패 케이스**: 선호도 불일치 시나리오
#### 생성 원칙
- **실제 패턴 기반**: 기존 282쌍의 성공 패턴 학습
- **다양성 확보**: 연령, 스타일, 선호도 조합多样化
- **품질 검증**: 생성 데이터의 현실성 검증

## 6. 연구적 의의 및 기대효과
### 6.1 학술적 기여
#### 이론적 가치
- **실증적 연구**: 이론이 아닌 실제 데이터 기반 매칭 연구
- **상호성 연구**: 일방적이 아닌 양방향 매력 분석
- **다차원 분석**: 시각, 인구통계, 심리적 요소 통합
#### 방법론적 혁신
- **삼중쌍 데이터셋**: 이미지-특징-이상형의 통합적 접근
- **성공 기준 재정의**: 대화가 아닌 상호 좋아요를 성공 기준으로
- **실제 검증**: 가상이 아닌 실제 매칭 성공 데이터
### 6.2 기술적 기대효과
#### 예측 정확도 향상
- **매력 예측**: 70% 이상의 상호 매력 예측 목표
- **선호도 학습**: 사용자 이상형 패턴 정확한 모델링
- **개인화 추천**: 개인별 최적 파트너 추천 시스템
#### 산업적 활용
- **매칭 알고리즘**: 현재 0.005% → 1% 이상 성공률 목표
- **UX 개선**: 사용자 만족도 및 앱 이탈률 개선
- **비용 효율**: 매칭 비용 감소 및 수익 증대
